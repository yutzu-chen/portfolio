Voice of Customer Intelligence System

Hundreds of customer reviews arrived weekly across 4 platforms — multilingual, fragmented, and largely unread.

Designed and built the system end-to-end, treating PMs as the primary users and designing for their real weekly workflow.

Built an LLM pipeline that aggregates reviews from Trustpilot, Google Play, App Store, and NPS — auto-translates, detects themes, runs sentiment analysis, and generates a structured weekly report. Added a searchable database and an AI chat for deeper exploration.

Python LLM / prompt design Trustpilot API App Store / Google Play Dashboard building Sentiment analysis Theme detection
  • 20k+ multi-channel reviews analyzed in one unified system
  • Weekly reporting fully automated — no manual work
  • Surfaced recurring bugs teams couldn't see when reviews were scattered
  • Enabled faster, evidence-backed product prioritization

Holidu: Conversion Uplift Through Experimentation

Funnel data showed friction — but root causes were unclear and stakeholder priorities were driven by intuition, not evidence.

AI & Automation Product Ops

Scaling Product Operations with Automation

Translation coordination, A/B test documentation, and Jira backlog noise consumed PM and engineering time that should have gone to actual product work.

Designed and built all four automation pipelines — from identifying the problem through implementation and rollout.

Built 4 automation systems: (1) OpenAI API → auto-upload translation pipeline, (2) Jira automation creating structured experiment docs on ticket status change, (3) Slack announcements on experiment launch, (4) lifecycle automation that closes stale tickets with no response.

Jira Automation OpenAI API n8n Slack API Workflow design Process improvement
  • ~80% reduction in translation processing time
  • Every experiment now has a structured, auto-generated record
  • Cleaner backlog — stale tickets no longer clog prioritization
  • Reduced repetitive support load for the whole team
